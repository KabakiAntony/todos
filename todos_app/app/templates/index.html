Todos

This is a todos REST API.

The following endpoints are available for use

  1. /users/signup [POST]
  2. /users/verify [POST]
  3. /users/signin [POST]
  4. /users/forgot [POST]
  5. /users/update-password [PUT]
  6. /todos [POST]
  7. /todos [GET]
  8. /todos/{id} [PUT]
  9. /todos/{id} [GET]
  10. /todos/{id} [DELETE]

This is for just whetting your appetite, for more information see the readme on github
